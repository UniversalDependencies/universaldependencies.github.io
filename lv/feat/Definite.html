Definite: definiteness or state

Latvian features gender for adjectives (ADJ) and ordinal numerals and some participles (VERB with VerbForm=Part).

Values used:

  • Ind (indefinite)
  • Def (definite)

Theoretically possible values:

  • Spec (specific indefinite) - in contemporary language ordinal numbers feature definite ending, e. g., trešais “third”, ceturtais “fourth”, however dictionaries and older teksts sometimes contain ordinal number forms with indefinite ending, e.g., trešs, ceturts. This is rare enough to be absent from corpus.

Values not present in Latvian:

  • Cons (construct state)
  • Com (complex)
