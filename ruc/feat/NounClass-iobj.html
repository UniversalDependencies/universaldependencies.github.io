NounClass[iobj]: noun class agreement with indirect object

Values: Bantu1 Bantu2

The indirect object iobj is a core argument of ditransitive predicates. It usually has the semantic role of a recipient or a goal. The indirect object can be marked on the verb as the second object prefix, following the subject prefix and the direct object prefix.

Third-person markers of the indirect object agree with the indirect object in noun class. The prefix form is identical to the one used in NounClass[obj]. First and second person prefixes of indirect objects do not show agreement in noun class; their forms cross-references Person[iobj] and Number[iobj].)

Bantu1: noun class 1

The indirect object prefix of noun class 1 is mu-. This class contains singular nouns with human reference. The corresponding plural noun class is Bantu2 and with some nouns Bantu10.

Examples

  • ba-ka-mu-wa
    NC2.SBJ-NC12.OBJ-NC1.IOBJ-give
    “they give it (money) to him”

Bantu2: noun class 2

The indirect object prefix of noun class 2 is ba-. This class contains plural nouns with human reference. The corresponding singular class is 1, as in Bantu1.

Examples

  • a-ka-ba-wa
    NC1.SBJ-NC12.OBJ-NC2.IOBJ-give
    “he gives it (money) to them”
