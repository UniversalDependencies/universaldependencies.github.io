NounClass[psed]: noun class agreement with the possessee

Values: Bantu1 Bantu2 Bantu3 Bantu4 Bantu5 Bantu6 Bantu7 Bantu8 Bantu9 Bantu10
Bantu11 Bantu12 Bantu13 Bantu14 Bantu15 Bantu22 Bantu23

In Ruuli, possessive pronouns agree both with the possessor and the possessee. The possessee agreement is marked by the initial morpheme of the possessive pronoun. Third person possessee marking reflects the noun class of the possessee: [NounClass[psed]], according to the paradigm below. Noun class agreement with the possessor is annotated with the feature NounClass[psor].

2 ba-

5 li-

15 ku-
20 gu-

The paradigm of possessee markers excludes locative classes 16, 17, 18, and 23 as they do not contain nouns and cannot be used as objects; Proto-Bantu noun classes 19 and 21 are not attested in Ruuli.

Bantu1: noun class 1

The possessee prefix of noun class 1 is wa-. This class contains singular nouns with human reference. The corresponding plural noun class is Bantu2 and Bantu10.

Examples

  • omwana wa-nge
    NC1.child NC1.PSED-1sg.PSOR
    “my child”

Bantu2: noun class 2

The possessee prefix of noun class 2 is ba-. This class contains plural nouns with human reference. The corresponding singular noun class is Bantu1.

Examples

  • abakulu ba-kyo
    NC2.leaders NC2.PSED-NC7.PSOR
    “its leaders”

Bantu3: noun class 3

The possessee prefix of noun class 3 is gu-, realized as gw- due to the following associative vowel. This class contains singular nouns. The corresponding plural class is Bantu4.

Examples

  • mugaso gw-agyo
    NC3.importance 3.PSED-4.PSOR
    “their importance”

Bantu4: noun class 4

The possessee prefixes of noun class 4 are gi-, realized as gy- due to the following associative vowel, and za-. This class contains plural nouns. The corresponding singular class is Bantu3.

Examples

  • emirimu gy-ange
    NC4.activities NC4.PSED-1sg.PSOR
    “my activities”

Bantu5: noun class 5

The possessee prefix of noun class 5 is li-, realized as ly- due to the following associative vowel. This class contains singular nouns. The corresponding plural class is Bantu6.

Examples

  • eibara ly-amu
    NC5.name NC5.PSED-2sg.PSOR
    “your name”

Bantu6: noun class 6

The possessee prefix of noun class 6 is ga-. This class contains plural and mass nouns. The corresponding singular class is Bantu5.

Examples

  • amaaba ga-abwe
    NC6.departure NC6.PSED-NC2.PSOR
    “their departure”

Bantu7: noun class 7

The possessee prefix of noun class 7 is ki-, realized as ky- due to the following associative vowel. This class contains singular nouns. The corresponding plural class is Bantu8.

Examples

  • kibanja ky-aiswe
    NC7.land NC7.PSED-1pl.PSOR
    “our land”

Bantu8: noun class 8

The possessee prefix of noun class 8 is bi-, realized as by- due to the following associative vowel. This class contains plural nouns. The corresponding singular class is Bantu7.

Examples

  • ebyojo by-abwe
    NC8.boys NC8.PSED-NC2.PSOR
    “their boys”

Bantu9: noun class 9

The possessee prefix of noun class 9 is ya-. This class contains singular nouns. The corresponding plural class is Bantu10.

Examples

  • entikko ya-agwo
    NC9.culmination NC9.PSED-NC3.PSOR
    “its culmination”

Bantu10: noun class 10

The possessee prefix of noun class 10 is za-. This class contains plural nouns. The corresponding singular classes are Bantu1, Bantu9, and Bantu11.

Examples

  • engeso za-alyo
    NC10.norms NC10.PSED-NC5.PSOR
    “its norms”

Bantu11: noun class 11

The possessee prefix of noun class 11 is lu-, realized as lw- due to the following associative vowel. This class contains singular nouns. The corresponding plural class is Bantu10.

Examples

  • olwetiiko lw-amu
    NC11.responsibility NC11.PSED-2sg.PSOR
    “your responsibility”

Bantu12: noun class 12

The possessee prefix of noun class 12 is ka-. This class contains singular and mass nouns. It is also used to form diminutive singular nouns. The corresponding plural class is Bantu14.

Examples

  • akawoowo ka-gyo
    NC12.scent NC12.PSED-NC9.PSOR
    “its scent”

Bantu13: noun class 13

The possessee prefix of noun class 13 is tu-, realized as tw- due to the following associative vowel. This class is used to form diminutive mass nouns. There is no corresponding plural class.

Examples

  • otusabbuuli tw-ange
    NC13.some_soap NC13-1sg.PSOR
    “my small amount of soap”

Bantu14: noun class 14

The possessee prefix of noun class 14 is bu-, realized as bw- due to the following associative vowel. This class contains multiple abstract nouns. It is also used to form diminutive plural nouns. The corresponding singular class is Bantu12.

Examples

  • obwomi bw-aiswe
    NC14.life NC14.PSED-1pl.PSOR
    “our life”

Bantu15: noun class 15

The possessee prefix of noun class 15 is ku-, realized as kw- due to the following associative vowel. This class contains deverbal nouns. There is no corresponding plural class.

Examples

  • okusoma kw-amu
    NC15.education NC15.PSED-2sg.PSOR
    “your education”

Bantu20: noun class 20

The possessee prefix of noun class 20 is gu-, realized as gw- due to the following associative vowel. This class is used to form augmentative singular nouns. The corresponding plural class is Bantu22.

Examples

  • ogubbambo gw-abwe
    NC20.big_basket NC20.PSED-NC2.PSOR
    “their big basket”

Bantu22: noun class 22

The possessee prefix of noun class 22 is ga-. This class is used to form augmentative plural nouns. The corresponding singular class is Bantu20.

Examples

  • agajwenge ga-bwe
    NC22.big_boats NC22.PSED-NC2.PSOR
    “their big boats”
