NounClass: noun class

Ruuli has a noun class system in which all nouns are assigned to one of 23 possible classes. The system also extends to other nominals (pronouns and proper nouns), as well as to agreeing parts of speech (adjectives, determiners, verbs, etc.). In addition, adpositions may reflect locative noun classes.

The Ruuli noun class system comprises 18 non-locative classes and four locative classes. No correspondences to Proto-Bantu classes 19 and 21 are attested. Noun class is primarily determined by the nominal prefix. As in many Bantu languages, nouns in certain singular classes (1, 3, 5, 7, 9, 11, 12, 22) typically form their plurals in corresponding classes (2, 4, 6, 8, 10, 14, 24).

Common singular–plural pairings include: 1/2, 1/10, 3/4, 5/6, 7/8, 9/10, 11/10, 12/14, and 20/22. Many nouns occur in only one class and lack either a singular or a plural counterpart.

Bantu1: noun class 1

The corresponding plural class is Bantu2 or Bantu10.

Examples

  • (o)mukoli “worker”
  • (o)fundi “tailor”
  • (o)njango “cat”

Bantu2: noun class 2

The corresponding singular class is Bantu1.

Examples

  • (a)bakoli “workers”
  • (a)bafundi “tailors”

Bantu3: noun class 3

The corresponding plural class is Bantu4.

Examples

  • (o)muganda “bundle”

Bantu4: noun class 4

The corresponding singular class is Bantu3.

Examples

  • (e)miganda “bundles”

Bantu5: noun class 5

The corresponding plural class is Bantu6.

Examples

  • (e)ibeere “breast”
  • (e)dinisa “window”
  • (e)riiso “eye”

Bantu6: noun class 6

The corresponding singular class is Bantu5.

Examples

  • (a)mabeere “breasts”
  • (a)madinisa “windows”
  • (a)maiso “eyes”

Bantu7: noun class 7

The corresponding plural class is Bantu8.

Examples

  • (e)kibbambo “bundle”

Bantu8: noun class 8

The corresponding singular class is Bantu7.

Examples

  • (e)bibbambo “bundles”

Bantu9: noun class 9

The corresponding plural class is Bantu10.

Examples

  • (e)nsansa “palm leaf”
  • (e)bbaasi “bus”

Bantu10: noun class 10

The corresponding singular class is Bantu1, Bantu9, or Bantu11.

Examples

  • (e)njango “cats”
  • (e)nsansa “palm leaves”
  • (e)bbaasi “buses”
  • (e)ndimi “flames”

Bantu11: noun class 11

The corresponding plural class is Bantu10.

Examples

  • (o)lulimi “flame”

Bantu12: noun class 12

Used to form diminutive nouns. The corresponding plural class is Bantu14.

Examples

  • (a)katiko “mushroom”
  • (a)kaana “small child” (from (a)mwana “child” Bantu1)

Bantu13: noun class 13

Used to form diminutive nouns. No corresponding plural class.

Examples

  • (o)tuceere “little rice” (from (o)muceere “rice” Bantu3)

Bantu14: noun class 14

Used to form diminutive nouns. The corresponding singular class is Bantu12.

Examples

  • (o)butiko “mushrooms”
  • (o)bwana “small children” (from (a)baana “children” Bantu2)

Bantu15: noun class 15

No corresponding plural class. Infinitives used as nouns ([VerbForm]=Vnoun) also belong to this class.

Examples

  • (o)kusaaga “joke”
  • (o)kubandula “marrying”

Bantu16: noun class 16

Locative noun class used in existential and locative constructions with the copula li.

Examples

  • Waliwo enjawulo. “There is a difference.”

Bantu17: noun class 17

Locative noun class denoting general location in space or time.

Examples

  • Ayabire oku igombolola. “He has gone _*to** the sub-county.”

Bantu18: noun class 18

Locative noun class denoting interiority.

Examples

  • Yaboine oitimba omu kisiko. “He saw a python _*in** the bush.”

Bantu20: noun class 20

Used to form augmentative nouns. The corresponding plural class is Bantu22.

Examples

  • (o)gusolo “large animal” (from (e)kisolo “animal” Bantu7)

Bantu22: noun class 22

Used to form augmentative nouns. The corresponding plural class is Bantu20.

Examples

  • (a)gasolo “large animals” (from (e)bisolo “animals” Bantu8)

Bantu23: noun class 23

Locative noun class, typically used with proper place names.

Examples

  • Twayaba e Kampala. “We will go _*to** Kampala.”
