Person[iobj]: person agreement with indirect object

Values: 1 2 3

The indirect object iobj is a core argument of ditransitive predicates and usually has the semantic role of a recipient or a goal. The indirect object can be marked on the verb as the second object prefix, following the subject prefix and the prefix of the direct object.

The prefix of the indirect object reflects its person.

1: first

First person singular object is marked as the object prefix n-, while first person plural object is marked as tu-.

Examples

  • a-ka-n-wa
    NC1.SBJ-NC12.OBJ-1sg.IOBJ-give
    “he gives it (money) to me”

  • a-ka-tu-wa
    NC1.SBJ-NC12.OBJ-1pl.IOBJ-give
    “he gives it (money) to us”

2: second

Second person singular object is marked as the object prefix ku-, while second person plural object is marked as ba-.

Examples

  • a-ka-ku-wa
    NC1.SBJ-NC12.OBJ-2sg.IOBJ-give
    “he gives it (money) to you (sg.)”

  • a-ka-ba-wa
    NC1.SBJ-NC12.OBJ-2pl.IOBJ-give
    “he gives it (money) to you (pl.)”

3: third

Third person object prefixes are realized by a paradigm of third person markers, known as noun classes. The finer-grained value of this marker is annotated as NounClass[iobj].

Examples

  • ba-ka-mu-wa
    NC2.SBJ-NC12.OBJ-NC1.IOBJ-give
    “they give it (money) to him”

  • a-ka-ba-wa
    NC1.SBJ-NC12.OBJ-NC2.IOBJ-give
    “he gives it (money) to them”
