NounClass[obj]: noun class agreement with object

Values: Bantu1 Bantu2 Bantu3 Bantu4 Bantu5 Bantu6 Bantu7 Bantu8 Bantu9 Bantu10
Bantu11 Bantu12 Bantu13 Bantu14 Bantu15 Bantu22 Bantu23

In Ruuli, objects are often marked on the verb as the object prefix, which follows the subject prefix (except for imperatives and infinitives, which do not have subject prefixes). Third-person object prefixes agree with the object in person, number, and – for the third person – in noun class, according to the mapping in the table below.

Noun Class (NC) Object Prefix
1 mu-
2 ba-
3 gu-
4 gi-
5 li-
6 ga-
7 ki-
8 bi-
9 gi-
10 zi-
11 lu-
12 ka-
13 tu-
14 bu-
15 ku-
20 gu-
22 ga-

The paradigm of object prefixes excludes the locative classes 16, 17, 18, and 23 as they do not contain nouns and cannot be used as objects. Proto-Bantu noun classes 19 and 21 are not attested in Ruuli.

Bantu1: noun class 1

The object prefix of the noun class 1 is _mu-_are This class contains singular nouns, mostly for humans. The corresponding plural noun classes are Bantu2 and, with some nouns, Bantu10.

Examples

  • A-mu-boine.
    NC1.SBJ-NC1.OBJ-see:PFV
    “He has seen him.”

Bantu2: noun class 2

The object prefix of noun class 2 is ba-. This class contains plural nouns with human reference. The corresponding singular class is Bantu1.

Examples

  • A-ba-boine.
    NC1.SBJ-NC2.OBJ-see:PFV
    “He has seen them (people).”

Bantu3: noun class 3

The object prefix of noun class 3 is gu-. This class contains singular nouns. The corresponding plural class is Bantu4.

Examples

  • Omwigo, a-gu-taire awo.
    NC3.stick NC1.SBJ-NC3.OBJ-put:PFV there
    “The stick, she has put it there.”

Bantu4: noun class 4

The object prefix of noun class 4 is gi-. This class contains plural nouns. The corresponding singular class is Bantu3.

Examples

  • Emisale tu-gi-malere.
    NC4.trees 1pl.SBJ-NC4.OBJ-finish:PFV
    “The trees, we did away with them.”

Bantu5: noun class 5

The object prefix of noun class 5 is li-. This class contains singular nouns. The corresponding plural class is Bantu6.

Examples

  • Eisumu n-li-zwireku.
    NC5.spear 1sg.SBJ-NC5.OBJ-leave:PFV
    “The spear, I have abandoned it.”

Bantu6: noun class 6

The object prefix of noun class 6 is ga-. This class contains plural and uncountable nouns. The corresponding singular class is Bantu5. This class contains plural and mass nouns. The corresponding singular class is Bantu5.

Examples

  • Amagezi gani tu-ga-taka.
    NC6.knowledge NC6.this 1pl.SBJ-NC6.OBJ-need
    “This knowledge, we need it.”

Bantu7: noun class 7

The object prefix of noun class 7 is ki-. This class contains singular nouns. The corresponding plural class is Bantu8.

Examples

  • Ekisunsuzo n-ki-waire maama.
    NC7.comb 1sg.SBJ-NC7.OBJ-give:PFV NC1.mother
    “The comb, I have given it to my mother.”

Bantu8: noun class 8

The object prefix of noun class 8 is bi-. This class contains plural nouns. The corresponding singular class is Bantu7.

Examples

  • Ebisolo tu-bi-tega.
    NC8.animals 1pl.SBJ-NC8.OBJ-hunt
    “The animals, we hunt them.”

Bantu9: noun class 9

The object prefix of noun class 9 is gi-. This class contains singular nouns. The corresponding plural class is Bantu10.

Examples

  • Ensonga eyo n-ku-gi-tegera.
    NC9.issue NC9.that 1sg.SBJ-PROG-NC9.OBJ-understand
    “That issue, I understand it.”

Bantu10: noun class 10

The object prefix of noun class 10 is zi-. This class contains plural nouns. The corresponding singular classes are Bantu1, Bantu9, and Bantu11.

Examples

  • Enyanja twzizwireku.
    NC10.lakes 1pl.SBJ-NC10.OBJ-leave:PFV
    “The lakes, we have given up on them.”

Bantu11: noun class 11

The object prefix of noun class 11 is lu-. This class contains singular nouns. The corresponding plural class is Bantu10.

Examples

  • Olwowo o-lu-boine?
    NC11.wooden_plate 2sg.SBJ-NC11.OBJ-see:PFV
    “The wooden plate, have you seen it?”

Bantu12: noun class 12

The object prefix of noun class 12 is ka-. This class contains singular and mass nouns. It is also used to form diminutive singular nouns. The corresponding plural class is Bantu14.

Examples

  • Ba-ka-sikaku akaita.
    NC2.SBJ-NC12.OBJ-fry NC12.millet
    “They fry it, the millet.”

Bantu13: noun class 13

The object prefix of noun class 13 is tu-. This class is used to form diminutive mass nouns. There is no corresponding plural class.

Examples

  • Otulo, n-ku-tw-endya.
    13.some_sleep 1sg.SBJV-PROG-NC13.OBJ-need
    “Some sleep, I need it.” (tw- is the surface form of the underlying tu-; the glide /w/ is conditioned by the following vowel)

Bantu14: noun class 14

The object prefix of noun class 14 is bu-. This class contains abstract nouns. It is also used to form diminutive plural nouns. The corresponding singular class is Bantu12.

Examples

  • Obudooli obwo ba-bu-gaine.
    NC14.small NC14.those NC2.SBJ-NC14.OBJ-refuse:PFV
    “Those small ones, they have refused them.”

Bantu15: noun class 15

The object prefix of noun class 15 is ku-. This class contains deverbal nouns. There is no corresponding plural class.

Examples

  • okusomesya, ti-n-kw-endya
    NC15.teaching NEG-1sg.SBJ-NC15.OBJ-like
    “Teaching, I don’t like it.” (kw- is the surface form of the underlying ku-; the glide /w/ is conditioned by the following vowel)

Bantu20: noun class 20

The object prefix of noun class 20 is gu-. This class is used to form augmentative singular nouns. The corresponding plural class is Bantu22.

Examples

  • ogubbambo, n-gu-bwoine
    NC20.big_basket 1sg.SBJ-NC20.OBJ-see:PFV
    “The big basket, I have seen it.”

Bantu22: noun class 22

The object prefix of noun class 22 is ga-. This class is used to form augmentative plural nouns. The corresponding singular class is Bantu20.

Examples

  • Agajwenge, n-ga-bwoine
    NC22.big_boats 1sg.SBJ-NC22.OBJ-see:PFV
    “The big boats, I have seen them.”
