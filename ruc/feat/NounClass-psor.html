NounClass[psor]: noun class agreement with the possessor

Values: Bantu1 Bantu2 Bantu3 Bantu4 Bantu5 Bantu6 Bantu7 Bantu8 Bantu9 Bantu10
Bantu11 Bantu12 Bantu13 Bantu14 Bantu15 Bantu22 Bantu23

In Ruuli, possessive pronouns agree both with the possessor and the possessee. The possessor agreement is marked by the final morpheme of the possessive pronoun. The third person possessor agreement cross-references the noun class of the possessor: [NounClass[psor]], according to the paradigm below. The noun class agreement with the possessee is annotated with the feature NounClass[psed].

The paradigm of the possessor markers excludes locative classes 16, 17, 18, and 23 as they do not contain nouns and are not used as possessors; The Proto-Bantu noun classes 19 and 21 are not attested in Ruuli.

Bantu1: noun class 1

The possessor marker of noun class 1 is mwe. This class contains singular nouns, mostly for humans. The corresponding plural classes are Bantu2 and Bantu10.

Examples

  • omwana wa-mwe
    NC1.child NC1.PSED-NC1.PSOR
    “his/her child”

Bantu2: noun class 2

The possessor marker of noun class 2 is bwe. This class contains plural nouns, mostly for humans. The corresponding singular class is Bantu1.

Examples

  • amaaba ga-bwe
    NC6.departure NC6.PSED-NC2.PSOR
    “their departure”

Bantu3: noun class 3

The possessor marker of noun class 3 is gwo. This class contains singular nouns. The corresponding plural class is Bantu4.

Examples

  • entikko ya-gwo
    NC9.culmination NC9.PSED-NC3.PSOR
    “its culmination” (e.g. of a function; mukolo ‘function (NC3)’)

Bantu4: noun class 4

The possessor marker of noun class 4 is gyo. This class contains plural nouns. The corresponding singular class is Bantu3.

Examples

  • mugaso gwa-gyo
    NC3.importance NC3.PSED-NC4.PSOR
    “their importance” (e.g. of jobs; milimo ‘jobs (NC4)’)

Bantu5: noun class 5

The possessor marker of noun class 5 is lyo. This class contains singular nouns. The corresponding plural class is Bantu6.

Examples

  • engeso za-lyo
    10.norms NC10.PSED-NC5.PSOR
    “its norms” (e.g. of a tribe; iyanga ‘tribe (NC5)’)

Bantu6: noun class 6

The possessor marker of noun class 6 is go. This class contains plural and mass nouns. The corresponding singular class is Bantu5.

Examples

  • ekikulu kya-go
    NC7.purpose NC7.PSED-NC6.PSOR
    “its purpose” (e.g. of butter; magita ‘butter (NC6)’)

Bantu7: noun class 7

The possessor marker of noun class 7 is kyo. This class contains singular nouns. The corresponding plural class is Bantu8.

Examples

  • abakulu ba-kyo
    NC2.leaders NC2.PSED-NC7.PSOR
    “its leaders” (e.g. of a clan; kika ‘clan (NC7)’)

Bantu8: noun class 8

The possessor marker of noun class 8 is byo. This class contains plural nouns. The corresponding singular class is Bantu7.

Examples

  • mwiiku gwa-byo
    NC3.scent NC3.PSED-NC8.PSOR
    “their scent” (e.g. of meals; biibulo ‘meals (NC8)’)

Bantu9: noun class 9

The possessor marker of noun class 9 is gyo. This class contains singular nouns. The corresponding plural class is Bantu10.

Examples

  • akawoowo ka-gyo
    NC12.scent NC12.PSED-NC9.PSOR
    “its scent” (e.g. of food; mere ‘food (NC9)’)

Bantu10: noun class 10

The possessor marker of noun class 10 is zo. This class contains plural nouns. The corresponding singular classes are Bantu1, Bantu9, and Bantu11.

Examples

  • amabara ga-zo
    NC6.names NC6.PSED-NC10.PSOR
    “their names” (e.g. of cows; nte ‘cows (NC10)’)

Bantu11: noun class 11

The possessor marker of noun class 11 is lwo. This class contains singular nouns. The corresponding plural class is Bantu10.

Examples

  • erangi ya-lwo
    NC9.color NC9.PSED-NC11.PSOR
    “its color” (e.g. of wood; lubaawo ‘wood (NC11)’)

Bantu12: noun class 12

The possessor marker of noun class 12 is ko. This class contains singular and mass nouns. It is also used to form diminutive singular nouns. The corresponding plural class is Bantu14.

Examples

  • ebiseera bya-ko
    NC8.season NC8.PSED-NC12.PSOR
    “its season” (e.g. of dry spell; kanyango ‘dry spell (NC12)’)

Bantu13: noun class 13

The possessor marker of noun class 13 is two. This class is used to form diminutive mass nouns. There is no corresponding plural class.

Examples

  • mugaso gwa-two
    NC3.importance NC3.PSED-NC13.PSOR
    “its importance” (e.g. of some sleep; tulo ‘some sleep (NC13)’)

Bantu14: noun class 14

The possessor marker of noun class 14 is bwo. This class contains multiple abstract nouns. It is also used to form diminutive plural nouns. The corresponding singular class is Bantu12.

Examples

  • omukama wa-bwo
    NC1.leader NC1.PSED-NC14.PSOR
    “its leader” (e.g. of the kingdom; bukama ‘kingdom (NC14)’)

Bantu15: noun class 15

The possessor marker of noun class 15 is kwo. This class contains deverbal nouns. There is no corresponding plural class.

Examples

  • amarakka ga-kwo
    NC6.sound NC6.PSED-NC15.PSOR
    “its sound” (e.g. of thunder; kubbwatuka ‘thunder (NC15)’)

Bantu20: noun class 20

The possessor marker of noun class 20 is gwo. This class is used to form augmentative singular nouns. The corresponding plural class is Bantu22.

Examples

  • omusaayi gwa-gwo
    NC3.blood NC3.PSED-NC20.PSOR
    “its blood” (e.g. of a large animal; gusolo ‘large animal (NC20)’)

Bantu22: noun class 22

The possessor marker of noun class 22 is go. This class is used to form augmentative plural nouns. The corresponding singular class is Bantu20.

Examples

  • omusaayi gwa-go
    NC3.blood NC3.PSED-NC22.PSOR
    “their blood” (e.g. of large animals; gasolo ‘large animals (NC22)’)
