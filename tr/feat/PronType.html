This page pertains to UD version 2.

PronType: pronominal type

This feature typically applies to pronouns, determiners, pronominal numerals (quantifiers) and pronominal adverbs.

Prs: personal

Examples

  • ben “I”
  • sen “you-SG”
  • siz “you-PL”

Rcp: reciprocal pronoun

This value covers reciprocal pronoun birbir- and its inflected forms.

Examples

  • Birbirinize iyi bakın “Take care of each other”
  • Akşamları birbirimize kitap okuruz “We read each other books in the evenings”

Int: interrogative pronoun, determiner, numeral or adverb

Examples:

  • kim “who”
  • ne “what”
  • kaç “how many”
  • kimin “whose”

Dem: demonstrative pronoun

Examples:

  • bu “this”
  • şu “that/this”
  • o “that”

Ind: indefinite pronoun

Examples:

  • bazı “some”

Loc: locative pronoun

(Not in UD)

Locative pronouns are typically distinguished from their demonstrative counterparts in Turkish.

Examples:

  • bura “here”
  • şura “there”
  • ora “there”

Neg: negative determiner or pronoun

This universal feature is used only a few Turkish treebanks for negative pronouns and determiners.

Examples:

  • hiç “none”
  • hiçbir “none”
  • hiçbiri “none of them”

Art: article

This universal feature is used only a few Turkish treebanks for article-like determiners, mainly for the indefinte determiner bir.

Examples:

  • bir “a”
