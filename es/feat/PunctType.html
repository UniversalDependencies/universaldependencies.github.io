PunctType: punctuation type

Values: Comm Peri Qest Brck Semi Colo Dash Quot Excl

Comm: Comma

This value is used for commas.

Examples:

  • Me gusta el pan, el queso y el vino.

Peri: Period

This value is used for periods.

Examples:

  • Terminé mi tarea.

Qest: Question Mark

This value is used for question marks.

Examples:

  • ¿Cómo estás?

Brck: Bracket

This value is used for brackets or parentheses.

Examples:

  • Estoy leyendo un libro (El principito).

Semi: Semicolon

This value is used for semicolons.

Examples:

  • Fui al parque; jugué fútbol.

Colo: Colon

This value is used for colons.

Examples:

  • Hay tres cosas que necesito: pan, leche y huevos.

Dash: Dash

This value is used for dashes.

Examples:

  • María -mi hermana- está en la casa.

Quot: Quotation Mark

This value is used for quotation marks.

Examples:

  • Él dijo: “Voy a la tienda”.

Excl: Exclamation Mark

This value is used for exclamation marks.

Examples:

  • ¡Hola!
