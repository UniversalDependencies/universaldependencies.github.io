Number: number

The Number feature for Basque follows the standard UD guidelines for nouns, adjectives, determiners and adverbs. However, finite verbs contain agreement features on number for the subject, object and indirect object, so the Basque treebank follows the UD description for language-specific features, defining Number[erg]=Sing,Plur, Number[abs]=Sing,Plur, and Number[dat]=Sing,Plur.

Person: person

The Person feature for Basque follows the standard UD guidelines for nouns, adjectives, determiners and adverbs. However, finite verbs contain agreement features on person for the subject, object and indirect object, so the Basque treebank follows the UD description for language-specific features, defining Person[erg]=1,2,3, Person[abs]=1,2,3, and Person[dat]=1,2,3.
