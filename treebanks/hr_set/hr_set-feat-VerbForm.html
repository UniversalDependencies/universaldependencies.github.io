Treebank Statistics: UD_Croatian-SET: Features: VerbForm

This feature is universal but the values Conv are language-specific. It occurs with 4 different values: Conv, Fin, Inf, Part.

32486 tokens (16%) have a non-empty value of VerbForm. 7350 types (21%) occur at least once with a non-empty value of VerbForm. 2414 lemmas (13%) occur at least once with a non-empty value of VerbForm. The feature is used with 4 part-of-speech tags: VERB (17418; 9% instances), AUX (12541; 6% instances), ADJ (2055; 1% instances), ADV (472; 0% instances).
