Treebank Statistics: UD_Croatian-SET: Features: VerbForm

This feature is universal but the values Conv are language-specific. It occurs with 4 different values: Conv, Fin, Inf, Part.

32479 tokens (16%) have a non-empty value of VerbForm. 7345 types (21%) occur at least once with a non-empty value of VerbForm. 2407 lemmas (13%) occur at least once with a non-empty value of VerbForm. The feature is used with 4 part-of-speech tags: VERB (17387; 9% instances), AUX (12566; 6% instances), ADJ (2054; 1% instances), ADV (472; 0% instances).
