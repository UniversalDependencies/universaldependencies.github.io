Treebank Statistics: UD_Croatian-SET: Features: Tense

This feature is universal. It occurs with 3 different values: Imp, Past, Pres.

26287 tokens (13%) have a non-empty value of Tense. 4760 types (13%) occur at least once with a non-empty value of Tense. 1914 lemmas (10%) occur at least once with a non-empty value of Tense. The feature is used with 4 part-of-speech tags: VERB (13791; 7% instances), AUX (12018; 6% instances), ADV (476; 0% instances), DET (2; 0% instances).
