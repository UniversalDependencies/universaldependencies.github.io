Treebank Statistics: UD_Pomak-Philotis: Features: Number

This feature is universal but the values Count are language-specific. It occurs with 4 different values: Coll, Count, Plur, Sing.

This is a layered feature with the following layers: Number, Number[psor].

18749 tokens (55%) have a non-empty value of Number. 6263 types (99%) occur at least once with a non-empty value of Number. 2842 lemmas (88%) occur at least once with a non-empty value of Number.
