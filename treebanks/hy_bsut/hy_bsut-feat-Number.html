Treebank Statistics: UD_Armenian-BSUT: Features: Number

This feature is universal but the values Assoc are language-specific. It occurs with 4 different values: Assoc, Coll, Plur, Sing.

This is a layered feature with the following layers: Number, Number[psor].

20175 tokens (44%) have a non-empty value of Number. 8245 types (70%) occur at least once with a non-empty value of Number. 3997 lemmas (61%) occur at least once with a non-empty value of Number.
