Treebank Statistics: UD_Old_East_Slavic-RNC: Features: Number

This feature is universal but the values Count are language-specific. It occurs with 4 different values: Count, Dual, Plur, Sing.

19011 tokens (54%) have a non-empty value of Number. 8510 types (89%) occur at least once with a non-empty value of Number. 4097 lemmas (85%) occur at least once with a non-empty value of Number.
