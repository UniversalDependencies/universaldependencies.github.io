Treebank Statistics: UD_Czech-PDT: Features: Gender

This feature is universal. It occurs with 3 different values: Fem, Masc, Neut. Some words have combined values of the feature; 3 combinations have been observed: Fem|Masc, Fem|Neut, Masc|Neut.

This is a layered feature with the following layers: Gender, Gender[psor].

160378 tokens (48%) have a non-empty value of Gender. 47347 types (89%) occur at least once with a non-empty value of Gender. 21569 lemmas (80%) occur at least once with a non-empty value of Gender. The feature is used with 8 part-of-speech tags: NOUN (79711; 24% instances), ADJ (40447; 12% instances), PROPN (14282; 4% instances), DET (10951; 3% instances), VERB (10879; 3% instances), AUX (1745; 1% instances), PRON (1397; 0% instances), NUM (966; 0% instances).
