Description

UD_Guajajara-TuDeT is a collection of annotated sentences in Guajajara. The project is a work in progress and the treebank is being updated on a regular basis. UD_Guajajara-TuDeT is a collection of annotated sentences in Guajajara. It is part of [TuLaR](https://tular.clld.org), Tupían Language Resources, which contains other languages -- see [TuDeT](https://tular.clld.org/contributions/tudet).
