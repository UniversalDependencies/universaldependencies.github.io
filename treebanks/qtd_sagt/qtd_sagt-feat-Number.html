Treebank Statistics: UD_Turkish_German-SAGT: Features: Number

This feature is universal. It occurs with 2 different values: Plur, Sing.

This is a layered feature with the following layers: Number, Number[psor].

17031 tokens (46%) have a non-empty value of Number. 5451 types (82%) occur at least once with a non-empty value of Number. 2703 lemmas (72%) occur at least once with a non-empty value of Number.
