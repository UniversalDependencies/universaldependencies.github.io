Treebank Statistics: UD_French-Spoken: Features: Person

This feature is universal. It occurs with 3 different values: 1, 2, 3.

This is a layered feature with the following layers: Person, Person[psor].

8372 tokens (24%) have a non-empty value of Person. 694 types (15%) occur at least once with a non-empty value of Person. 334 lemmas (10%) occur at least once with a non-empty value of Person. The feature is used with 4 part-of-speech tags: PRON (4177; 12% instances), VERB (2262; 6% instances), AUX (1535; 4% instances), DET (398; 1% instances).
