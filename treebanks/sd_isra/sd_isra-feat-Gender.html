Treebank Statistics: UD_Sindhi-Isra: Features: Gender

This feature is universal. It occurs with 2 different values: Fem, Masc.

This is a layered feature with the following layers: Gender, Gender[obj], Gender[subj].

6684 tokens (42%) have a non-empty value of Gender. 2002 types (66%) occur at least once with a non-empty value of Gender. 1656 lemmas (72%) occur at least once with a non-empty value of Gender.
