Description

This is a treebank annotating example sentences from a comprehensive grammar book of Turkish.

This treebank was originally annotated with the aim of setting guidelines for the Turkish UD. All fields are annotated manually for UD v1. The conversion to v2 included some automatic changes. Further information on the treebank and the annotation process can be found at the the treebank web page.

References

  • The treebank is described in, Çağrı Çöltekin (2015) A grammar-book treebank of Turkish In: Proceedings of the 14th workshop on Treebanks and Linguistic Theories (TLT 14)
  • The example sentences are from Aslı Göksel and Celia Kerslake. Turkish: A Comprehensive Grammar. London: Routledge, 2005.
