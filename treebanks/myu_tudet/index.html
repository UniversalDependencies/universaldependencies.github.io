Description

UD_Munduruku-TuDeT is a collection of annotated sentences in Mundurukú. The project is a work in progress and the treebank is being updated on a regular basis.

</br> </br> </br> </br> </br>

UD_Munduruku-TuDeT is a collection of annotated sentences in Mundurukú. It is part of TuLaR, Tupían Language Resources, which contains other languages – see TuDeT. The sentences are being annotated by Fabrício Ferraz Gerardi and Eva Huber.
