Treebank Statistics: UD_Serbian-SET: Features: VerbForm

This feature is universal but the values Conv are language-specific. It occurs with 4 different values: Conv, Fin, Inf, Part.

15992 tokens (16%) have a non-empty value of VerbForm. 4066 types (22%) occur at least once with a non-empty value of VerbForm. 1440 lemmas (15%) occur at least once with a non-empty value of VerbForm. The feature is used with 4 part-of-speech tags: VERB (8413; 9% instances), AUX (6203; 6% instances), ADJ (1094; 1% instances), ADV (282; 0% instances).
