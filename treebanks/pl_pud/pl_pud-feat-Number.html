Treebank Statistics: UD_Polish-PUD: Features: Number

This feature is universal. It occurs with 3 different values: Plur, Ptan, Sing.

This is a layered feature with the following layers: Number, Number[psor].

10848 tokens (59%) have a non-empty value of Number. 7134 types (95%) occur at least once with a non-empty value of Number. 4589 lemmas (92%) occur at least once with a non-empty value of Number.
