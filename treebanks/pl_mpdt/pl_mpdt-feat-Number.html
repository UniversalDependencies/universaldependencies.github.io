Treebank Statistics: UD_Polish-MPDT: Features: Number

This feature is universal. It occurs with 4 different values: Dual, Plur, Ptan, Sing.

This is a layered feature with the following layers: Number, Number[psor].

24536 tokens (52%) have a non-empty value of Number. 12784 types (94%) occur at least once with a non-empty value of Number. 6584 lemmas (87%) occur at least once with a non-empty value of Number.
