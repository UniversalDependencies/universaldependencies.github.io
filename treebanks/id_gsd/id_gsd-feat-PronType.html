Treebank Statistics: UD_Indonesian-GSD: Features: PronType

This feature is universal but the values Emp are language-specific. It occurs with 8 different values: Art, Dem, Emp, Ind, Int, Prs, Rel, Tot.

10573 tokens (9%) have a non-empty value of PronType. 151 types (1%) occur at least once with a non-empty value of PronType. 90 lemmas (1%) occur at least once with a non-empty value of PronType. The feature is used with 4 part-of-speech tags: PRON (6417; 5% instances), DET (3664; 3% instances), ADV (404; 0% instances), NUM (88; 0% instances).
