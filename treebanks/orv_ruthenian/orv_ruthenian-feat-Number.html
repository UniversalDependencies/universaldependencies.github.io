Treebank Statistics: UD_Old_East_Slavic-Ruthenian: Features: Number

This feature is universal but the values Count are language-specific. It occurs with 4 different values: Count, Dual, Plur, Sing.

5009 tokens (50%) have a non-empty value of Number. 2606 types (83%) occur at least once with a non-empty value of Number. 1133 lemmas (77%) occur at least once with a non-empty value of Number.
