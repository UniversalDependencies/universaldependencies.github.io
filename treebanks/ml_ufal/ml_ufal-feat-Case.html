Treebank Statistics: UD_Malayalam-UFAL: Features: Case

This feature is universal but the values Cmp are language-specific. It occurs with 7 different values: Acc, Cmp, Dat, Gen, Ins, Loc, Nom.

81 tokens (40%) have a non-empty value of Case. 62 types (49%) occur at least once with a non-empty value of Case. 51 lemmas (48%) occur at least once with a non-empty value of Case. The feature is used with 3 part-of-speech tags: NOUN (36; 18% instances), PRON (33; 16% instances), PROPN (12; 6% instances).
