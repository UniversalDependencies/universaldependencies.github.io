Treebank Statistics: UD_Slovak: Features: Gender

This feature is universal. It occurs with 3 different values: Fem, Masc, Neut.

This is a layered feature with the following layers: Gender, Gender[psor].

52564 tokens (50%) have a non-empty value of Gender. 23391 types (90%) occur at least once with a non-empty value of Gender. 11915 lemmas (84%) occur at least once with a non-empty value of Gender.
