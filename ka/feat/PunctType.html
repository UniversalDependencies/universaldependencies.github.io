PunctType: punctuation type

Values: Peri Comm Brck Dash Excl Qest Colo Semi Elip Quot

The PunctType feature applies to punctuation.

Peri: dot

Examples

  • .

Comm: comma

Examples

  • ,

Brck: parenthesis

Examples

  • ()

Dash: hyphenation

Examples

  • -

Excl: exclamation

Examples

  • !

Qest: interrogation

Examples

  • ?

Colo: colon

Examples

  • :

Semi: semicolon

Examples

  • ;

Elip: ellipsis

Examples

  • …

Quot: colon

Examples

  • “
