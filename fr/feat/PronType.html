PronType: pronominal type

French uses the following values for `PronType’: definite (Def), indefinite (Ind), negative (Neg), personal (Prs), relative (Rel) and interrogative (Int).

Not all relative and interrogative pronouns are marked so far.

Treebank Statistics (UD_French)

This feature is universal. It occurs with 6 different values: Dem, Ind, Int, Neg, Prs, Rel.

25775 tokens (6%) have a non-empty value of PronType. 130 types (0%) occur at least once with a non-empty value of PronType. 34 lemmas (0%) occur at least once with a non-empty value of PronType. The feature is used with 2 part-of-speech tags: fr-pos/PRON (15492; 4% instances), fr-pos/DET (10283; 3% instances).
