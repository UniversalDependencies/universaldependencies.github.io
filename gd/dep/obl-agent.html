obl:agent: passive agent oblique nominal

The obl:agent relation is used for a nominal (noun, pronoun, noun phrase) that specifies the agent in a passive construction using the irregular verb rach and a verbal noun. They are marked by the adposition le. It works together with aux:pass, which relates the irregular verb acting as an auxiliary to the verbal noun and nsubj:pass which specifies the passive subject.
